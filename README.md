# Password Strength Checker (Java)

## Description
This project checks the strength of a password based on length, uppercase letters, lowercase letters, numbers, and special characters.

## Technologies Used
- Java

## Features
- Password length validation
- Uppercase, lowercase, digit, and special character check
- Displays password strength as Weak, Medium, or Strong

## How to Run
1. Run the Java program
2. Enter a password when prompted
3. View the strength result
